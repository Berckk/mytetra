app - Создать bat-файл для запуска скрипта в указанном каталоге

Параметры:

<ИмяСкрипта> - Имя скрипта в текущем каталоге или полный путь скрипта

<Каталог> - (необязательно) Каталог, в котором будет создан скрипт запуска. По умолчанию "C:\Program Files (x86)\OneScript\bin"

-name - Имя генерируемого исполняемого файла
