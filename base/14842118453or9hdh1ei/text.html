Для firefox с установленным дополнением VimFx

1. Перейти в режим чтения (gr)

2. Скопировать содержимое (Ctr-a, Ctr-c)

3. Перейти в myTetra

4. Создать новую статью

5. Вставить (Ctrl-v) содержимое буфера обмена в тело статьи.


Для chrome с плагином EasyReader

* нажать на клавишу плагина

* выделить нужную часть статьи

* копировать

* перейти в myTetra

* создать новую статью

* вставить содержимое буфера обмена
