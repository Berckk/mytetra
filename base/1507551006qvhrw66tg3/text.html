В рамках этого урока познакомимся с объектами HTTPСоединение, HTTPЗапрос. Для понимания работы с указанными объектами сделаем модуль обновления курсов валют.

Урок можно разбить на следующие этапы:

  1. Коррекция структуры и данных для обновления курсов валют;
  2. Создание Общего модуля обрабатывающего обновление валют;
  3. Создание обработки обновления курсов;
  4. Создание тестов на базе xUnitFor1C для проверки правильности работы обновления курсов валют.

1. Коррекция структуры и данных для обновления курсов

Добавим возможность указания необходимости обновления курсов из интернет:

  • В справочник Валюты добавим реквизит «ОбновлятьИзИнтернет», тип Булево;
  • В регистр сведений Курсы валют добавим ресурс Кратность (число 10);
  • В регистре сведений Курсы валют изменим точность курса, укажем: Число 10, 4;
  • В общей форме «ФормаЭлементаВалюты» добавим отображение реквизита ОбновлятьИзИнтернет;

Для валюты Доллар США (USD) укажем код 840, для Евро 978, у обоих валют установим признак ОбновлятьИзИнтернет в Истина.

/// Скачивает курс валюты из интернет

///

/// @param ИмяСервера - Строка - Домен с курсами

/// @param ПутьКФайлуНаСервере - Строка - Адрес с курсом валюты

///

/// @return Строка, Неопределено - Строка если курс скачен, иначе Неопределено

///

Функция СкачатьКурсИзИнтернет(ИмяСервера, ПутьКФайлуНаСервере)

 

    Соединение = Новый HTTPСоединение(ИмяСервера);    

 

    HTTPЗапрос = Новый HTTPЗапрос(ПутьКФайлуНаСервере);

    HTTPЗапрос.Заголовки.Вставить("Accept-Charset", "utf-8");

 

    Ответ = Соединение.Получить(HTTPЗапрос);

 

    //# Код 200 - успешное равершение

    Если Ответ.КодСостояния <> 200 Тогда

        Возврат Неопределено;

    КонецЕсли;

 

    //# Получим результат в виде строки, минуя временные файлы.

    Возврат Ответ.ПолучитьТелоКакСтроку();    

 

КонецФункции

 

 

/// Загрузить курс валюты за указанный период

///

/// @param ДатаНачала - Дата - Начало периода получаемых курсов

/// @param ДатаОкончания - Дата - Конец периода получаемых курсов

/// @param Валюта - Справочник.Валюты - Валюта, для которой получаем курс

///

/// @return Булево - Истина - Получилось обновить курсы, иначе Ложь

///

Функция ЗагрузитьКурсВалюты(ДатаНачала, ДатаОкончания, Валюта) Экспорт

 

    СерверИсточник = "cbrates.rbc.ru";

 

    Если ДатаНачала = ДатаОкончания Тогда

        Адрес = "tsv/";

        ТМП   = Формат(ДатаОкончания, "ДФ=/yyyy/MM/dd");

    Иначе

        Адрес = "tsv/cb/";

        ТМП   = "";

    КонецЕсли;

 

    ПутьКФайлуНаСервере = Адрес + Прав(Валюта.Код, 3) + ТМП + ".tsv";

 

    СтрокиКурса = СкачатьКурсИзИнтернет(СерверИсточник, ПутьКФайлуНаСервере);

 

    Если СтрокиКурса = Неопределено Тогда

        Возврат Ложь;

    КонецЕсли;

 

    КолСтрок = СтрЧислоСтрок(СтрокиКурса);

    ДатаКурса = ДатаОкончания;

 

    Для НомерСтроки = 1 По КолСтрок Цикл

 

        Стр = СтрПолучитьСтроку(СтрокиКурса, НомерСтроки);

 

        Если (Стр = "") ИЛИ (СтрНайти(Стр, Символы.Таб) = 0) Тогда

            Продолжить;

        КонецЕсли;

 

        МассивСтрок = СтрРазделить(Стр, Символы.Таб, Истина);

        Если МассивСтрок.Количество() = 2 Тогда

            Кратность = Число(МассивСтрок[0]);

            Курс = Число(МассивСтрок[1]);

        Иначе

            ДатаКурсаСтр = МассивСтрок[0];

            ДатаКурса = Дата(Лев(ДатаКурсаСтр,4), Сред(ДатаКурсаСтр,5,2), Сред(ДатаКурсаСтр,7,2));

            Кратность = Число(МассивСтрок[1]);

            Курс = Число(МассивСтрок[2]);

        КонецЕсли;

 

        Если ДатаКурса < ДатаНачала Тогда Продолжить; КонецЕсли; Если ДатаКурса > ДатаОкончания Тогда

            Продолжить;

        КонецЕсли;

 

        НаборЗаписей = РегистрыСведений.КурсыВалют.СоздатьНаборЗаписей();

        НаборЗаписей.Отбор.Валюта.Установить(Валюта);

        НаборЗаписей.Отбор.Период.Установить(ДатаКурса);

 

        Запись = НаборЗаписей.Добавить();

        Запись.Валюта = Валюта;

        Запись.Период = ДатаКурса;

        Запись.Курс = Курс;

        Запись.Кратность = Кратность;

 

        НаборЗаписей.Записать();

 

    КонецЦикла;

 

    Возврат Истина;

 

КонецФункции

4. Создание тестов на базе xUnitFor1C для проверки

правильности работы обновления курсов валют.

Проверка работоспособности функционала является важной частью разработки. Не всегда можно проверить правильность работы кода простой пробежкой глазами по данным. Реализуем проверку работы функционала скачивания и разбора курса валют используя фреймворк xUnitFor1C.

Важно: В уроке используется 4 версия, редакция подправленная мной, позволяющая работать в режиме отключенной модальности. На момент написания статьи создан Pull Request на внесение доработки в основной репозиторий в ветку dev.
