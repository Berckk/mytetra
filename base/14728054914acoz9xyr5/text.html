## Работа с регистрами

# Типы регистров

* ""

* регистр по-умолчанию. В него помещается любой скопированный или удаленный текст

* 0

* содержит последний текст, который был скопирован (y) без указания регистра

* 1

* содержит послдений удаленный текст, при условии, что при удалении не был указан регистр и текст содержит хотя бы одну полную строку.

# Команды работы с регистрами

* отобразить список регистров

:reg

* вставить содержимое из регистра

"{имя регистра}p

* вставить содержимое из регистра %

"%p

* перейти в каталог текущего файла

:cd %:p:h

* поместить содержимое регистра % в регистр *

:let @*=@%

* удалить текущую строку в регистр "a"

"aD

* скопировать д:ве строки в регист "a"

"a2Y

* удалить и дописать текущую строку в регистр "a"

"Add

* вставить содержимое регистра "a" после или перед текущей строкой

"a[pP]

* вставить содержимое регистра в режиме вставки

<Ctrl+r>[reg_name]
