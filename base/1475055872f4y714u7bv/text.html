Пример: Есть таблица с полями "Контрагент", "Договор". В списке выбора договора должны быть только договора из строк таблицы с определенным контрагентом.

&НаСервере

Функция ПолучитьСписокВыбораДоговораКонтрагентаИзРеестраПлатежей(Контрагент)

СписокВыбора = Новый СписокЗначений;
