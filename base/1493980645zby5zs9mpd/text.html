Как работать с LaTeX

Простейший документ

Создавать документы с помощью LaTeX просто и легко. В отличие от Word, работа начинается с плоского текста, который содержит код LaTeX и наполнение документа (т.е. текст). LaTeX использует контрольные указатели, которые определяют формат содержания документа. Простейший документ может быть создан с помощью следующего кода:

\documentclass{article}
\begin{document}
  Hello, World!
\end{document}
