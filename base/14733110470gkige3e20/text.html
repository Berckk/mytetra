# Добавление плагинов с помощью vundle

* поместить между операторами начала и окончания вызова vundle оператор с указанием на git-репозиторий плагина

call vundle#begin()

Plugin ‘pola/my-cool-vim-plugin’

call vundle#end()

* перезапустить vim и перечитать _vimrc

:so $MYVIMRC

* установить плагин

:PluginInstall

## Удаление плагина с помощью vundle

* удалить оператор с указанием на git-репозиторий плагина

*вызвать команду

:PluginClean!
