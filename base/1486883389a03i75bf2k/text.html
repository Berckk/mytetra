Plugin 'majutsushi/tagbar' " Class/module browser

файл .ctags - отвечает за поиск тех самых функций и процедур в файле. Нужен для правильной работы с файлом программы Exuberant ctags (http://ctags.sourceforge.net/index.html).
Вот tagbar и ctags работают совместно.
У tagbar есть настройки различные

" TagBar настройки

map <F4> :TagbarToggle<CR>

let g:tagbar_autofocus = 0 " автофокус на Tagbar при открытии

Настройки выше позволят открывать окно tagbar по нажатию F4.

Переход к нужной строке можно усуществить, например

  • переключится на окно с tagbar Ctrl-W
  • выбрать строку с нужной функцией или процедурой и нажать Enter
    Так вы окажитесь на нужной строке.

Выполнить команду в :!ctags file.bsl,чтобы создать файл меток для вашего файла.

  • в каталоге с файлом будет создан файл tags.
  • выполнить команду в редакторе :tag имя тэга
    Так вы тоже попадёте на необходимую строку. При вводе тэга поддерживается автодополнение по нажатию клавиши TAB.
