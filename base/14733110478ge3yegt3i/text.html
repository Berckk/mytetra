# Введение

Для автоматизации работы и записи повторяющихся действий можно использовать макросы. Запись введенных команд производится непосредственно в именованные регистры.

Важным моментом при работае с макросами является то, что существующий макрос можно расширить. Также как и в случае с именованными регистрами, можно использовать заглавную

букву для добавления действий к уже имеющимся.


# Некоторые команды

* q{a-z} начать запись макроса в указанный именованный регистр

* q окончить запись макроса

* @{a-z} воспроизвести макрос, сохранненый в указанном регистре

* <Ctrl+r>{a-z} выполнить макрос в режиме вставки

* qaq очистить регистр a

* qaf,r0@aq рекурсивно вызвать макрос замены всех символов , символом 0

* "ap вставить в текущий буфер содержимое регистра a
