Ошибка при старте процесса

{БизнесПроцесс.МойПроцесс.Форма.ФормаБизнесПроцессаОбычная.Форма(26)}: Ошибка при вызове метода контекста (Старт)

БизнесПроцессОбъект.Старт();

по причине:

Значение "УУ-071104" поля "Номер" не уникально


Исправление:

в консоли кода выполнить ОбновитьНумерациюОбъектов(Метаданные.Задачи.ЗадачаИсполнителя);
