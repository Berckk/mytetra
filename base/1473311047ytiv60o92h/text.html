## Навигация в vim

* отобразить текущий каталог

:pwd

* перейти в директорию текущего файла

:cd %:p:h

* перейти в директорию только для текущего окна

:lcd %:p:h


* описание команд

* %

* возвращает имя текущего файла

* %:p

* возвращает полный путь текущего файла

* %:p:h

* возвращает путь директории текущего файла


| action | command |

------------------------------

| PageDown | Ctrl-B |

| PageUp | Ctrl-F |

| 1\2 PageDown | Ctrl-D |

| 1\2 PageUp | Ctrl-U |

| Down | Ctrl-E |

| Up | Ctrl-Y |

| H, M, L | |
