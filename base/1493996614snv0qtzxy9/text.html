Вариант с установкой pandoc на компьютер

Предварительно установить pandoc (см. статью в /soft/converters/pandoc)

  1. Создать файл с расширением test1.md и кодировкой
  2. Наполнить файл используя разметку markdown
  3. Сохранить файл. Если файл в кодировке windows-1251, то изменить кодировку на utf-8. Если текстовый редактор vim, то это можно сделать так:
    • set
  4. Открыть cmd в каталоге файла
  5. Выполнить команду > pandoc test1.md -f markdown -t html -s -o test1.html
  6. Открыть файл test1.html > test1.html
  7. Открыть html-содержимое командой и скопировать содержимое
  8. Создать пустую статью в mytetra
  9. Перейти в html-редактор статьи
  10. Заменить содержимое

Вариант с использованием pandoc-online

  1. Копировать текст файла md
  2. Вставить в левую часть online конвертера
  3. Нажать convert
  4. Скопировать html-содержимое правой части
  5. Создать пустую ветку в mytetra
  6. Перейти в html-редактор статьи
  7. Заменить содержимое
