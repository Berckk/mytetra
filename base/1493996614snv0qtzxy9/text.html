Предварительно установить pandoc (см. статью в mytetra)

1. Создать файл с расширением test1.md и кодировкой

2. Наполнить файл используя разметку markdown

3. Сохранить файл.

Если файл в кодировке windows-1251, то изменить кодировку на utf-8.

Если текстовый редактор vim, то это можно сделать так:

* <F8>

* set <utf-8>

4. Открыть cmd в каталоге файла

5. Выполнить команду

> pandoc test1.md -f markdown -t html -s -o test1.html

6. Открыть файл test1.html

> test1.html

7. Открыть html-содержимое командой <Ctrl+U> и скопировать содержимое

8. Создать пустую статью в mytetra

9. Перейти в html-редактор статьи

10. Заменить содержимое
