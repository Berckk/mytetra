Предварительно установить pandoc (см. статью в mytetra)

1. Создать файл с расширением test1.md

2. Наполнить файл используя разметку markdown

3. Сохранить файл

4. Открыть cmd в каталоге файла

5. Выполнить команду

> pandoc test1.md -f markdown -t html -s -o test1.html

6. Открыть файл test1.html с помощью текстового редактора

7. Скопировать содержимое

8. Создать пустую статью в mytetra

9. Перейти в html-редактор статьи

10. Заменить содержимое
