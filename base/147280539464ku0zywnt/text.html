# Работа с вкладками

* добавитть новый буфер

:tabnew[file]

:tabe[file]

* закрыть текущую вкладку или под номером №

:tabc[N]

* закрыть все вкладки, кроме текущей

:tabo

* перейти на первую вкладку

:tabfir

* перейти на последнюю вкладку

:tabl

* перейти на предыдущую

:tabp

* перейти на следующую

:tabn [N]

* переместить текущую на место после N-ой вкладки

:tabm [N]

* показать список вкладок с номерами и открытыми файлами

:tabs

* выполнить команду [cmd] во всех вкладках

:tabdo {cmd}

* поместить все открытие файлы (буферы) во вкладки

:tab ball

* переключиться между буферами

:[N]gt

:c-tab

* закрыть буфер

:tac

:q

# Работа с окнами

* добавить новое окно

:new

* добавить вертикальное окно

:vnew

* переключиться между окнами

:c-ww

:c-w-(h,l)

:c-tab
