### Для чего предназначена

Программа относится к классу pim-менеджеров, т.е. программ для накопления и

хранения персональной информации.

Таких программ много:

* бесплатные

* Cherrytree

* zim

* Outwiker

* платные

* evernote

Используются для хранения заметок, паролей, организации списка дел. Можно

хранить важные документы и файлы. Все это есть в mytetra.

### Обзор интерфейса

Интерфейс классический

* слева дерево разделов

* вверху список записей каждого раздела

* внизу область редактирования записей

* внизу есть область поиска

### Действия над записями и ветками дерева раздела

* добавление

* редактирование (свойств, текста записи). Запись выполняется автоматически.

Запись имеет атрибуты: название, автор, url, метки.

Закрыть окно добавления или редактирования можно комбо Ctrl+Enter.

* перемещение записи с помощью кнопок со стрелками. Например, для

упорядочивания списка дел.

* можно вырезать записи в буфер обмена и переносить в другие ветки

* множественный выбор записей через контекстное меню. Далее можно скопировать,

удалить. Перенести (drag&drop).
